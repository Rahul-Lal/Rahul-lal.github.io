## Overview
This website serves as a portfolio to display some of the projects I've worked on. It provides detailed information about each project, including the project description, live demo, and GitHub repository. 

Feel free to browse the website to explore my work, and don't hesitate to reach out if you're interested in collaborating or hiring!

## Technologies Used
- HTML5
- CSS3 (including Flexbox and Grid)
- JavaScript (Vanilla JS)
- JSON (for dynamic project loading)
- GitHub Pages (for deployment)

## Features
- **Responsive Design:** Fully responsive layout that works well on both desktop and mobile devices.
- **Dynamic Project Listing:** Projects are loaded dynamically from a `projects.json` file.
- **Smooth Navigation:** Interactive navigation bar that takes you to different sections like "About," "Projects," and "Contact."
- **Project Details:** Each project has a description, a link to the live demo, and a link to the GitHub repository.
