A tale of HTML & CSS










What are HTML & CSS

HTML (HyperText Markup Language) and CSS (Cascading Style Sheet) are the fundamental building blocks to websites. HTML focuses on what's on the page which are known as elements. While CSS focuses on what the elements looks like, the format they present, elements that derive from the HTML, it makes no sense having a CSS file without a HTML page to connect it to.

Their relationship

Like Batman & Robin, HTML & CSS are a dyanamic duo, both are different on how they present themselves, but work together as one.

Another way of see this

I see the two languages working as one, as thinking of not as website, but as a car. The HTML representing the components of an automobile such as the exterior panels, how many seats, where the cup-holders, how much space in the car boot, etc. CSS focuses on the more finer details such as what color is the car, what kind of seats, are the windows tinted, type of wheels/tyres, etc.
